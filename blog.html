What is DOM and what is the purpose of DOM?

Document object model (DOM) is a tree of elements built by the browser.

Developers are use the Document Object Model (DOM) to create and manage documents, also navigate structure and add, change or delete elements.

How to select HTML elements using DOM? Name of the DOM methods

We can select HTML elements using 5 DOM Methods mentioned below:

  1. etElementsByTagName()
  2. getElementsByClassName()
  3. getElementById()
  4. querySelector()
  5. querySelectorAll()

What is event bubble?

Event Bubbling is an event that bubbles up from the target or the innermost parts to its parents, then takes a bottom to top approach and moves the event control flow to past events.

Event bubbling is the default event flow technique in the latest browsers.

What is a callback function and why we use it?

A callback function is a function that is given as an argument to another function. The function that gets the callback function as an argument is often called a higher-order function. When the higher-order function is called, it can execute the callback function given as input.

Callback function is used for code flexibility and reusability.
